# Student Placement Analysis Dashboard

## 📊 Project Overview
This project uses **Power BI** to analyze student data and predict placement outcomes.  
The dashboard provides interactive visualizations to explore key factors influencing placements such as academic performance, skills, and demographics.

## 📂 Repository Contents
- **placement analysis dashboard.pbix** → Power BI dashboard file  
- **student data for placement prediction.xlsx.csv** → Dataset used for analysis  
- **student placement dashboard (1).png** → Screenshot of the dashboard  
- **student placement dashboard (modified) (2).png** → Updated screenshot  

## 🚀 How to Use
1. Download the `.pbix` file.  
2. Open it in **Power BI Desktop**.  
3. Load the dataset (`.xlsx.csv`) if prompted.  
4. Explore the dashboard visuals and filters.

## 🔍 Insights
- Placement trends based on academic scores.  
- Comparison of students placed vs. not placed.  
- Key predictors of successful placement.  
- Visual breakdown of demographics and skill sets.

## 🛠️ Future Improvements
